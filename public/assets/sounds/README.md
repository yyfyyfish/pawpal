# PawPal Sound Cues

Bundled cues are intentionally short, local, and low-volume.

## Sources

- `meow-soft.wav` is trimmed and normalized from BigSoundBank sound #0494,
  "Little Meow of a Cat #1" by Joseph SARDIN, released as CC0/public domain.
  Source page: https://bigsoundbank.com/little-meow-of-a-cat-s0494.html
- `purr-short.wav` is trimmed and normalized from BigSoundBank sound #0436,
  "Cat Purr" by Joseph SARDIN, released as CC0/public domain.
  Source page: https://bigsoundbank.com/cat-purr-s0436.html
- `scratch-soft.wav` is a locally generated soft scratch texture. It does not
  use third-party source audio.

The source files were converted to mono 24 kHz WAV and faded/attenuated for a
desktop pet UI.
